Semantic HTML

Explanation

Semantic HTML uses tags that describe the meaning of the content. This helps browsers, developers, and search engines understand the page structure.

Example

<header>
Website Title
</header>

<nav>
Home | HTML | CSS
</nav>

<main>

<section>
<h2>HTML Basics</h2>
<p>HTML is used to create web pages.</p>
</section>

</main>

<footer>
Copyright 2026
</footer>

My Notes

header → top of the page nav → navigation menu main → main content section → content section footer → bottom of the page
