HTML Headings

HTML headings are used to define titles and subtitles on a webpage. They range from h1 to h6.

Examples

This is Heading h1

This is Heading h2

This is Heading h3

This is Heading h4

This is Heading h5
This is Heading h6

h1 is the most important heading and usually used for the main title of a page.
