HTML Page Structure

Explanation

Modern HTML pages are structured using semantic elements. The most common structure includes header, nav, main, and footer.

Example

<header>
Site Title
</header>

<nav>
Home | HTML | CSS
</nav>

<main>
Page Content
</main>

<footer>
Copyright
</footer>

It will look like this in the browser:

Site Title

Home | HTML | CSS
Page Content
Copyright

My Notes

Modern HTML pages are structured using semantic elements.

Common page structure:

header → top part of the page
nav → navigation menu
main → main content of the page
section → groups related content
article → independent content
footer → bottom part of the page

Important

nav is often placed inside the header because the navigation menu is usually at the top of the page.

main contains the primary content of the page that users come to see.

footer represents the bottom part of the page and is normally separate from the header.

section groups related content.

article represents independent content such as a blog post or tutorial.

An article is often placed inside a section.
