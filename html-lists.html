HTML Lists

Explanation

HTML lists are used to group related items together. There are two main types: ordered lists and unordered lists.

Example

<ul>
  <li>Apple</li>
  <li>Banana</li>
  <li>Orange</li>
</ul>

It will look like this in the browser:

  • Apple
  • Banana
  • Orange

My Notes

The<ul>tag creates an unordered list. The <li> tag defines each list item.
