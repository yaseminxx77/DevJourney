HTML Attributes

Explanation

HTML attributes provide additional information about elements. They are always written inside the opening tag.

Example

<a href="https://google.com">Visit Google</a>

<img src="cat.jpg" alt="Cat" width="200">

It will look like this in the browser:

Visit Google

My Notes

Attributes give extra information to HTML elements.

href → link destination
src → image source
alt → image description
class → used for styling
id → unique element identifier
