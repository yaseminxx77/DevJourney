HTML id vs class

Explanation

The class attribute is used to group multiple elements. The id attribute is used to identify a unique element.

Example

<div id="header">DevJourney</div>

<div class="product">Product 1</div>
<div class="product">Product 2</div>
<div class="product">Product 3</div>

It will look like this in the browser:

DevJourney
Product 1
Product 2
Product 3

My Notes

class → used for multiple elements
id → unique element

CSS:
.class → class selector
#id → id selector

class → used for groups of elements
example: A class / B class

<div class="classA">Ali</div>
<div class="classA">Mehmet</div>

Both students belong to the same group (classA).

id → used for unique elements
each element must have a different id.

<div id="student1">Ali</div>
<div id="student2">Mehmet</div>
