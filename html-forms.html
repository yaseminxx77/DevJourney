HTML Forms

Explanation

HTML forms are used to collect user input. Common examples include login forms, signup forms, and contact forms.

Example

<form>

<label>Name</label>
<input type="text">

<label>Email</label>
<input type="email">

<button>Submit</button>

</form>

My Notes

Forms are used to collect user input.

Important elements:
form → container for input fields
label → describes the input field
input → the field where the user enters data
button → submits the form

Common input types:

text → normal text input
email → email address input
password → hidden password input
number → input for numbers
checkbox → multiple choice selection
radio → select one option from a group
date → select a date

More form elements:

textarea → used for longer text input
select → dropdown list
option → choices inside a select element
