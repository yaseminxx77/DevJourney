HTML Div and Span

Explanation

The div tag is used to group larger sections of content. The span tag is used to style small parts of text.

Example

<div>
<h2>HTML Section</h2>
<p>Learning HTML.</p>
</div>

<p>This is <span style="color:red;">important</span> text.</p>

It will look like this in the browser:

HTML Section

Learning HTML.

This is important text.
