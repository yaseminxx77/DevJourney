HTML Tables

Explanation

HTML tables are used to display data in rows and columns.

Example

<table border="1">
<tr>
<th>Name</th>
<th>Age</th>
</tr>

<tr>
<td>Ali</td>
<td>22</td>
</tr>

<tr>
<td>Ayşe</td>
<td>21</td>
</tr>
</table>

It will look like this in the browser:

Name Age
Ali 22
Ayşe 21

My Notes

The table tag creates the table. The tr tag defines a row. The th tag creates a header cell and the td tag defines table data.
table → creates the table
tr → creates a row
th → header cell (column title)
td → data cell
